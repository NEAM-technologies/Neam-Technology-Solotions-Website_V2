Software Developer

  • Software Development:
    • Design, code, test, and debug software applications using relevant programming languages and tools.
    • Collaborate with cross-functional teams to define software requirements and specifications.
  • Problem Solving:
    • Analyze complex technical problems and propose effective solutions.
    • Debug and troubleshoot issues to ensure optimal software performance.
  • Software Architecture:
    • Contribute to the design and architecture of software systems.
    • Implement best practices for scalability, security, and maintainability.
  • Collaboration:
    • Work closely with other developers, designers, and product managers to ensure project success.
    • Participate in code reviews and provide constructive feedback.
  • Documentation:
    • Create and maintain technical documentation, including code documentation and user guides.
  • Continuous Learning:
    • Stay up-to-date with industry trends, emerging technologies, and best practices in software development.
    • Continuously improve your skills and share knowledge with the team.

Requirements:

  • Proven experience as a Software Developer in a professional setting.
  • Proficiency in one or more programming languages (e.g., HTML, Python, CSS, React/React Native, JavaScript).
  • Strong problem-solving skills and attention to detail.
  • Familiarity with software development methodologies (e.g., Agile, Scrum).
  • Excellent teamwork and communication skills.
  • Knowledge of version control systems (e.g., GitHub, Jira) and development tools.
  • Experience with open-source projects is a plus.

Why Join NEAM:

  • Engage in challenging and impactful software projects.
  • Collaborate with a diverse and talented team of professionals.
  • Competitive compensation and benefits package.
  • Opportunities for professional growth and career advancement.
  • A culture of innovation and a commitment to excellence.
